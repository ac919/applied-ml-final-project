## Data

Use Git LFS to pull data: should see 11 total csvs:
- games.csv
- players.csv
- plays.csv
- week1.csv -> week8.csv (8 weeks of player tracking data)

## Using Google Colab
Recommend to run this project in Google Colab in order to utilize their GPU.
- Open `run.ipynb` directly from GitHub: 
**File → Open Notebook → GitHub tab** → paste `https://github.com/ac919/applied-ml-final-project`
- Change runtime type -> T4 GPU
- run each cell of the notebook, which calls all the necessary scripts for data processing, model training, and evaluation
