## Resources
This project was heavily inspired by a submission to the 2025 NFL Big Data Bowl titled "Exposing Coverage Tells in the Presnap," which used transformers to predict man vz zone coverage. This project includes a similar version of their man-zone transformer, as well as a new extension: an additional transformer model for multiclass coverage classification. The key differences between this project's man-zone transformer & the one from "Exposing Coverage Tells in the Presnap" is the data it was trained on: whereas they included some frames of post snap data I focused solely on presnap data (before the play starts).

The data from this project comes from the 2023 NFL Big Data Bowl (the 2025 data is no longer publicly available online). The dataset contains 8 weeks of player tracking data from the 2023 NFL season. More info can be found [here](https://www.kaggle.com/competitions/nfl-big-data-bowl-2023/data).

## AI Use
